# # STUDY / LIFE Conductor

電車・バスの運転士みたいに「到着（完了）→次タスクへ」を押して進めるタスク運用アプリです。
GitHub Pagesでそのまま動きます（ビルド不要、HTML/CSS/JSのみ）。

## 使い方
- 「勉強線」「生活線」を切り替え
- タスクを追加
- 右の運転席で「発車」→カウントダウン
- 終わったら「到着（完了）」で次へ

## GitHub Pages 公開手順（最短）
1. このフォルダをGitHubにpush
2. Settings → Pages
3. Branch: main / (root) を選択して Save
4. 表示されたURLにアクセス

## 保存
localStorageに保存します（同じブラウザ/端末なら残ります）。study-conductor
